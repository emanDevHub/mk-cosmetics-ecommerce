# MK Cosmetics E-Commerce Website

A modern and responsive cosmetics e-commerce website developed as an academic subject project. The project provides a user-friendly online shopping interface with product listings, shopping features, and responsive web design using front-end technologies.

## Features
1. Responsive Website Design  
2. Product Display Section  
3. Shopping Cart Interface  
4. User-Friendly Navigation  
5. Modern UI Design  
6. Cosmetics Product Showcase  

## Technologies Used
1. HTML5  
2. CSS3  
3. JavaScript  
4. Bootstrap  

## Purpose
This project was created for learning and educational purposes to improve front-end web development and website design skills.
